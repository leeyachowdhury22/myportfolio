My Projects

Kettle

Kettle is a responsive Web App designed for people with food intolerance and dietary restriction. I completed both the UX and UI processes which include: competitor’s analysis, user research, wireframes, rapid prototyping and mock-ups. Conducting UX research gave this project a solid foundation to build on which then enabled me to design functional and meaningful UI.

Opal

From a money saving design brief, I created a mobile app that allows users to keep track of their finances as well as creating goals and budgeting. By conducting user research, user flow diagram, low, mid, high-fidelity wireframes, user testing, and lastly using the results for user interface design. For this project, I also created a set of brand guidelines in order to implement effective branding throughout the app.
