Kettle

Project Overview

Kettle is a responsive web app designed for people with food intolerance or allergies. 

Purpose & Context

This project was apart of the CareerFoundry UI design program. For this particular achievement we were asked to create a responsive web app.

Retrospective

what went well?

Conducting research through a user-first approach was a great way to keep the project focused and to validate specific features and ideas created for my recipe app.

What could be better?

As this was my first responsive web-app I found that there were a lot of hurdles to overcome. During the design process, I far too long making sure everything was pixel perfect which then led to burning out and creative blocks.

Solution

Spend some time getting inspired before diving into the design process, set small milestones and take regular breaks.
